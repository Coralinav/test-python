These are tests for python.

# Installation

1. Clone repo
2. Install and run pew for the virtual environment

```bash
pip install pew  
pew new test-python
pew workon
```

3. Install the `requirements.txt`

```bash
pip install -r .\requirements.txt
```
4. Now you can run the tests

example
```bash
python find-broken-links.py
```
